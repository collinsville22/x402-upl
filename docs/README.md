# X402-UPL Documentation

This directory contains the complete documentation for the X402 Universal Payment Layer platform.

## Documentation Structure

- [**index.md**](index.md) - Main documentation homepage
- [**architecture.md**](architecture.md) - Complete system architecture
- [**setup.md**](setup.md) - Installation and setup guide

## Viewing Documentation

### Online (GitHub Pages)
Visit: https://collinsville22.github.io/x402-upl/

### Locally
1. Install dependencies: `npm install`
2. Build documentation: `npm run docs:build`
3. Serve locally: `npm run docs:serve`

## Contributing to Documentation

Documentation is written in Markdown and hosted via GitHub Pages with Jekyll.

To update documentation:
1. Edit the relevant .md file in this directory
2. Commit changes
3. Push to master branch
4. GitHub Pages will automatically rebuild
